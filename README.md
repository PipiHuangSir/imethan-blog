ImEthan the3 web project

============================

使用的开源框架如下：
1、采用Spring作为IOC容器
2、使用Spring Data JPA接口规范，底层为hibernate实现
3、使用Spring MVC作为表现层实现框架
4、采用Apache Shiro作为安全和权限控制
5、采用bootstrap作为页面风格
6、使用sitemesh作为页面装饰框架
7、使用Jquery作为脚本语言
8、使用的数据库是MySQL 5
9、使用Gradle为项目的集成
10、使用JqGrid作为表格渲染
11、使用Ztree作为树形展现
